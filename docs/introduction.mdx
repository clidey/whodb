# Welcome to WhoDB

WhoDB is a powerful, intuitive database management tool designed to simplify how you interact with your databases. Whether you're a developer, data analyst, or database administrator, WhoDB provides a streamlined interface for exploring schemas, managing data, and visualizing relationships.

![WhoDB Login Page](/images/01-login-page.png)

## What is WhoDB?

WhoDB is an open-source database management solution that combines the power of traditional database clients with modern visualization and query capabilities. It provides a unified interface for working with multiple database types while maintaining a focus on simplicity and user experience.

## Key Features

<AccordionGroup>
<Accordion title="Database Connectivity">
WhoDB supports multiple database engines out of the box. Connect to your databases using standard connection strings or configure advanced options like SSL certificates, SSH tunnels, and connection pooling.
</Accordion>
<Accordion title="Schema Explorer">
Navigate your database structure with ease. The Schema Explorer shows all your databases, schemas, tables, views, and stored procedures in an organized tree view. Click any table to instantly view its structure and data.
</Accordion>
<Accordion title="Interactive Data Grid">
View and edit data in a familiar spreadsheet-like interface. Sort columns, filter rows, search across all fields, and perform bulk operations with just a few clicks.
</Accordion>
<Accordion title="Scratchpad Query Editor">
Write SQL queries in a powerful editor with syntax highlighting, auto-completion, and multi-cell support. Execute queries, view results, and save frequently used queries for later use.
</Accordion>
<Accordion title="Graph Visualization">
Visualize your database schema as an interactive graph. See table relationships, foreign keys, and data flow at a glance. Pan, zoom, and click nodes to explore your database structure visually.
</Accordion>
<Accordion title="Mock Data Generation">
Generate realistic test data for development and testing. Configure the number of rows, choose between append or overwrite modes, and let WhoDB populate your tables with sample data.
</Accordion>
<Accordion title="Export Capabilities">
Export your data in multiple formats including CSV, Excel, JSON, and SQL. Choose specific columns, apply filters, and select custom delimiters for maximum flexibility.
</Accordion>
</AccordionGroup>

## Use Cases

<Tip>
WhoDB is perfect for a wide range of database management tasks
</Tip>

**Development & Testing**
- Quickly inspect database state during development
- Generate test data for your applications
- Debug SQL queries and view results instantly
- Explore schema changes and relationships

**Data Analysis**
- Export data for further analysis in Excel or other tools
- Run ad-hoc queries to answer business questions
- Filter and sort large datasets to find specific records
- Visualize table relationships to understand data flow

**Database Administration**
- Monitor table structures and indexes
- Manage user data and permissions
- Backup and restore data using export/import
- Maintain data quality with filtering and validation

**Learning & Education**
- Explore database structures to learn SQL
- Visualize relationships between tables
- Practice queries in a safe environment
- Understand database design patterns
